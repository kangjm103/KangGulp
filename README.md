# gulp

> 초기 작업
> 1. node.js 필요
> 2. npm install 하면 package.json 을 참고하여 install 해줌

> package.json 없을 시, 직접 만들고 의존성 추가 해줄 경우
> 1. npm init
> 2. npm i -D gulp gulp-babel gulp-uglify babel-core babel-preset-es2015 gulp-strip-debug

> 실행
> gulp
